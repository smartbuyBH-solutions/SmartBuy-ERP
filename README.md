# SmartBuy-ERP

Interface oficial do Sistema Smart Buy BH e Ordens de Serviço.

## Responsabilidades

- App Shell e navegação;
- páginas e componentes React;
- Design System;
- formulários;
- acessibilidade;
- apresentação dos dados;
- consumo da API oficial do SmartBuy-Backend.

## Governança

- código integralmente nativo;
- código legado não pode ser incorporado;
- arquivos manuais limitados a 300 linhas;
- God Objects, God Components e God Screens são proibidos;
- regras comerciais não podem ser executadas como autoridade no frontend;
- acesso direto às tabelas operacionais é proibido;
- alterações devem ocorrer por Pull Request e testes.
